# 🎮 Projeto AOC - Jogo de Cartas em Assembly

![Badge Assembly](https://img.shields.io/badge/Language-Assembly%20MIPS-red)
![Badge Status](https://img.shields.io/badge/Status-Produção-red)

> Projeto desenvolvido para a disciplina de **Arquitetura e Organização de Computadores (AOC)**. O objetivo é implementar um jogo de cartas funcional utilizando a linguagem Assembly e manipulação de arquivos.

---

## 📋 Sobre o Projeto

Este repositório contém o código fonte de um jogo desenvolvido em baixo nível. O programa interage com o usuário e realiza a leitura de arquivos externos para definir o baralho ou estado do jogo.

### 📂 Estrutura dos Arquivos

- **`Jogo.asm`**: Código fonte principal contendo a lógica do jogo.
- **`cartas.txt`**: Arquivo de texto contendo os dados das cartas (naipes/valores) que são lidos pelo programa.
- **`Regras.txt`**: Documento descrevendo a lógica e as regras aplicadas no jogo.
- **`convenções.txt`**: Documentação sobre o uso dos registradores e padrões adotados no código.

---

## 🛠️ Tecnologias Utilizadas

- **Assembly MIPS**: Linguagem utilizada para toda a lógica.
- **MARS (MIPS Assembler and Runtime Simulator)**: Simulador recomendado para executar o código.
